# Commands

## `/dig`

Main command. Rummages through recent chats and proposes a haul.

```bash
/dig
```

Use it when you want Trash Panda to rescue fresh artifacts.

## `/den`

Shows the den dashboard.

```bash
/den
```

Use it when you want to glance at open tasks, projects, recent hauls, and the inbox.

## `/sniff`

Dry run. Shows what Trash Panda would find, but writes nothing.

```bash
/sniff
```

Good first try.

## `/feed <chat>`

Point Trash Panda at one specific chat.

```bash
/feed <chat>
```

## `/wash`

Tidies the den and collapses duplicates.

```bash
/wash
```

Raccoons wash their food. Obviously.

## `/nap`

Pause auto-digs.

```bash
/nap
```
