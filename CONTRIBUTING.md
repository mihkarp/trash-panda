# Contributing to Trash Panda 🦝

Trash Panda is small on purpose: a local-first skill that rescues useful artifacts from messy AI chats.

The best contributions make the raccoon either:

1. **more useful** — better extraction, routing, dedupe, den format
2. **more trustworthy** — clearer backlinks, privacy, approvals
3. **more shareable** — better Wrapped badges, sample hauls, examples
4. **more raccoon** — better copy, jokes, tiny rituals

## Good first issues

- Add a new artifact type
- Add examples to `examples/sample-den/`
- Improve `docs/chatgpt-export.md`
- Add issue templates
- Improve `/dig` approval copy
- Add more Trash Panda Wrapped archetypes

## Artifact taxonomy

Current core artifacts:

| Artifact | Meaning |
|---|---|
| 📝 To Do’s | concrete actions the user said they would do |
| 💎 Shiny | ideas worth revisiting |
| 🗺️ The Plan | structured strategy, route, next steps |
| 📜 Loot | reusable drafts, specs, prompts, notes |

A good artifact should be actionable or reusable. Trash Panda should not save generic summaries.

## Contribution style

Keep it:

- local-first
- markdown-first
- low ceremony
- funny, but useful
- transparent about source chats

Avoid:

- SaaS assumptions
- hidden writes
- vague summaries
- too many artifact categories
- “AI memory” buzzword soup

## Pull requests

Small PRs are easiest to review.

Please include:

- what changed
- why it helps
- a before/after example if possible

Trash Panda appreciates shiny things, but he appreciates clean diffs more.
